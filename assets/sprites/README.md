# Included Texture Pack

This folder now ships with a complete starter **SVG texture pack** so the game runs with real assets immediately.

## Included categories
- `terrain/`: snow, rock, ice tiles
- `buildings/`: bunker/shack parts and doors
- `items/`: resources, ammo, meds, food/water, money
- `characters/`: player, scav, arm layer
- `weapons/`: pistol, revolver, smg, rifle, shotgun, sniper
- `ui/`: crosshair
- `effects/`: muzzle flash, impact, blood

These are intentionally simple and editable. Replace any file with your own art using the same filename to keep game code unchanged.
